# 项目简介

本系统是基于BERT深度学习与知识图谱技术构建的法律罪名预测及问答平台。其主要用途是辅助法律工作者在案件分析过程中进行罪名预测和解答法律相关问题。功能模块包括罪名预测、法律问答和知识图谱构建。技术特点在于运用了预训练的BERT模型，以实现更准确的文本理解和预测。系统同时提供了详尽的文档说明和全部相关资料，以便使用者更好地理解与运用。


## 核心功能
- 罪名预测：辅助法律工作者在案件分析过程中进行罪名预测
- 法律问答：解答法律相关问题
- 知识图谱构建：提供法律领域的知识图谱功能

## 适用场景
- 法律案件分析
- 法律问题咨询
- 法律研究与学习
- 法律文书起草辅助

## 技术特点
- 基于BERT深度学习技术实现文本理解和预测
- 集成知识图谱技术构建法律领域知识体系
- 提供完整的源码、文档说明及全部相关资料
- 预训练模型支持更准确的文本理解和预测结果
